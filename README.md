Usee VTubing App
Мобильное приложение для виртуального ютубинга (VTubing), позволяющее анимировать 3D-аватар в реальном времени с помощью камеры устройства.

Возможности:
- Загрузка VRM-аватаров — поддержка стандарта VRM с автоматической конвертацией в GLB
- Real-time трекинг — отслеживание позы (33 точки) и лица через ML Kit
- Фото и видео — захват снимков и запись видео с 3D-аватаром
- Две камеры — поддержка фронтальной и задней камеры с зеркалированием
- Сохранение моделей — загруженный аватар сохраняется между запусками

Требования:
- Android 8.0 (API 26) или выше
- Разрешения: CAMERA, READ/WRITE_EXTERNAL_STORAGE

Использование:
1. На экране персонажа загрузите .vrm файл
2. Перейдите на экран камеры
3. Разрешите доступ к камере
4. Двигайтесь — аватар повторяет ваши движения
5. Используйте кнопки для фото и записи видео
